## Бизнес-требования

Приложение выполняет следующие бизнес-функции:
- Регистрация клиента
- Аутентификация клиента
- Заказы
    - Принятие заказа
    - Оплата заказа
- Нотификация клиента об изменении статуса заказа
- Склад
    - Пополнение склада
    - Изменение информации о товаре
- Доставка
    - Назначение курьера на доставку
    - Доставка заказа клиенту
